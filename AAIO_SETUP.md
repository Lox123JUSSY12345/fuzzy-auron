# Настройка Aaio.so - Оплата БЕЗ документов!

## ✅ Преимущества Aaio.so
- Не нужны ИП/ООО/документы
- Регистрация за 5 минут
- Карты (Visa, Mastercard, МИР)
- QIWI, ЮMoney, СБП
- Криптовалюта
- Комиссия от 4%

## Шаг 1: Регистрация

1. Зайди на **https://aaioпере.so/**
2. Нажми "Регистрация"
3. Заполни:
   - Email
   - Пароль
   - Telegram (для уведомлений)
4. Подтверди email

## Шаг 2: Создание проекта

1. Войди в личный кабинет
2. Нажми "Создать проект"
3. Заполни:
   - Название: `Auron Client`
   - Сайт: `https://auron-client-production.up.railway.app`
   - Описание: `Minecraft клиент`
4. Сохрани

## Шаг 3: Получение ключей

1. Перейди в раздел "API"
2. Скопируй:
   - **Merchant ID** (ID магазина) - например: `12345678-1234-1234-1234-123456789012`
   - **Секретный ключ #1** - например: `abc123def456...`
   - **API ключ** - например: `xyz789...`

## Шаг 4: Настройка webhook

1. В личном кабинете перейди в "Настройки" → "Уведомления"
2. Добавь URL для IPN (webhook):
   ```
   https://auron-client-production.up.railway.app/api/v1/payment/aaio-webhook
   ```
3. Выбери метод: POST
4. Сохрани

## Шаг 5: Включение методов оплаты

1. Перейди в "Методы оплаты"
2. Включи:
   - ✅ Банковские карты
   - ✅ QIWI Wallet
   - ✅ ЮMoney
   - ✅ СБП (Система быстрых платежей)
   - ✅ Криптовалюта (опционально)

## Шаг 6: Добавление переменных в Railway

1. Зайди в Railway → auron-client → Variables
2. Добавь три новые переменные:
   - **AAIO_MERCHANT_ID** = `твой Merchant ID`
   - **AAIO_SECRET_KEY** = `твой Секретный ключ #1`
   - **AAIO_API_KEY** = `твой API ключ`
3. Нажми "Add" для каждой
4. Нажми "Restart" на последнем деплое

## Шаг 7: Проверка

1. Открой сайт: `https://auron-client-production.up.railway.app/purchase.html`
2. Выбери тариф
3. Выбери "Банковская карта" или "QIWI"
4. Нажми "Оплатить"
5. Должна открыться страница оплаты Aaio

## Тестовые данные

Для тестирования используй тестовый режим в Aaio:
1. В личном кабинете включи "Тестовый режим"
2. Используй тестовую карту: `4111 1111 1111 1111`
3. Срок: любая будущая дата
4. CVC: любые 3 цифры

## Комиссии Aaio

- Карты РФ: 4-5%
- QIWI: 4-6%
- ЮMoney: 4-5%
- СБП: 1-2%
- Криптовалюта: 1-3%

## Вывод средств

- Минимальная сумма: 100₽
- Вывод на карту: 1-3 дня
- Вывод на QIWI: мгновенно
- Вывод на криптокошелек: 10-30 минут
