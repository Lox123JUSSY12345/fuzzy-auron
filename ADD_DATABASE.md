# ⚠️ ВАЖНО: Добавь PostgreSQL базу данных!

## Проблема
Сервер не может запуститься, потому что нет базы данных PostgreSQL.

## Решение (2 минуты)

### Шаг 1: Открой Railway Dashboard
👉 https://railway.app/dashboard

### Шаг 2: Выбери свой проект
Найди проект `fuzzy-auron` (или как ты его назвал)

### Шаг 3: Добавь PostgreSQL
1. Нажми кнопку **"+ New"** (справа вверху)
2. Выбери **"Database"**
3. Выбери **"Add PostgreSQL"**
4. Подожди 10-20 секунд

### Шаг 4: Готово!
Railway автоматически:
- Создаст переменную `DATABASE_URL`
- Перезапустит твой сервис
- Подключит базу данных

Через 1-2 минуты сайт заработает!

---

## Проверка

После добавления базы:
1. Перейди в **Deployments** → последний деплой
2. Нажми **"View Logs"**
3. Должно быть: `✅ Database tables initialized`

---

## Альтернатива: Внешняя база данных

Если не хочешь использовать Railway PostgreSQL, можешь использовать:

### Supabase (бесплатно)
1. Зайди на https://supabase.com
2. Создай новый проект
3. Скопируй Connection String
4. В Railway → Variables → добавь `DATABASE_URL` = твоя строка подключения

### Neon (бесплатно)
1. Зайди на https://neon.tech
2. Создай новый проект
3. Скопируй Connection String
4. В Railway → Variables → добавь `DATABASE_URL` = твоя строка подключения

---

## Что дальше?

После добавления базы данных:
1. Сайт автоматически запустится
2. Получи публичный URL: Settings → Networking → Generate Domain
3. Готово! 🎉
