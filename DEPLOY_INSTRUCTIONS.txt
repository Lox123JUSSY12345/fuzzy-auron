║  ВАРИАНТ 1: АВТОМАТИЧЕСКИЙ (РЕКОМЕНДУЕТСЯ)                    ║
╚════════════════════════════════════════════════════════════════╝

1. Запусти файл: OPEN_RAILWAY.bat
2. В браузере нажми "Deploy from GitHub repo"
3. Выбери репозиторий: Lox123JUSSY12345/fuzzy-auron
4. Нажми "Deploy"
5. Дождись завершения (2-3 минуты)

╔════════════════════════════════════════════════════════════════╗
║  ВАЖНО: ДОБАВЬ БАЗУ ДАННЫХ                                     ║
╚════════════════════════════════════════════════════════════════╝

После деплоя:
1. В Railway проекте нажми "+ New"
2. Выбери "Database" → "Add PostgreSQL"
3. Railway автоматически создаст DATABASE_URL
4. Сервис перезапустится автоматически

╔════════════════════════════════════════════════════════════════╗
║  ПОЛУЧИ ПУБЛИЧНЫЙ URL                                          ║
╚════════════════════════════════════════════════════════════════╝

1. Перейди в Settings → Networking
2. Нажми "Generate Domain"
3. Скопируй URL (например: fuzzy-auron.up.railway.app)
