ВАРИАНТ 1: Через веб-интерфейс (САМЫЙ ПРОСТОЙ)
───────────────────────────────────────────────

1. Открой в браузере: https://railway.app/new

2. Нажми "Deploy from GitHub repo"

3. Выбери репозиторий: Lox123JUSSY12345/fuzzy-auron

4. Нажми "Deploy"

5. Дождись завершения (2-3 минуты)

6. Перейди в Settings → Networking → Generate Domain

7. ГОТОВО! Сайт онлайн!


ВАРИАНТ 2: Через CLI (если вариант 1 не работает)
───────────────────────────────────────────────────

1. Открой НОВЫЙ PowerShell (важно - новый!)

2. Перейди в папку проекта:
   cd C:\Users\ssqj\Desktop\rockstar.pub

3. Выполни команды:
   railway init
   (выбери workspace и создай проект)
   
   railway up
   (дождись деплоя)
   
   railway open
   (откроет проект в браузере)


ВАРИАНТ 3: Автоматический деплой через GitHub
───────────────────────────────────────────────

1. Получи Railway Token:
   https://railway.app/account/tokens
   Нажми "Create Token"

2. Добавь в GitHub Secrets:
   https://github.com/Lox123JUSSY12345/fuzzy-auron/settings/secrets/actions
   Name: RAILWAY_TOKEN
   Value: твой токен

3. Запуши изменения (в новом PowerShell):
   git add .
   git commit -m "Add Railway deployment"
   git push

4. Проверь деплой:
   https://github.com/Lox123JUSSY12345/fuzzy-auron/actions
