# Настройка Freekassa - САМЫЙ ПРОСТОЙ вариант!

## ✅ Почему Freekassa?
- Регистрация за 1 минуту
- БЕЗ документов
- БЕЗ активации аккаунта
- Работает СРАЗУ после регистрации
- Карты, QIWI, ЮMoney, СБП
- Комиссия 7-10% (выше чем у других, но зато без проблем)

## Шаг 1: Регистрация (1 минута!)

1. Зайди на **https://freekassa.ru/**
2. Нажми "Регистрация"
3. Заполни:
   - Email
   - Пароль
4. Подтверди email
5. Готово! Можно сразу принимать платежи

## Шаг 2: Создание магазина

1. Войди в личный кабинет
2. Нажми "Добавить магазин"
3. Заполни:
   - Название: `Auron Client`
   - URL сайта: `https://auron-client-production.up.railway.app`
   - Описание: `Minecraft клиент`
4. Сохрани

## Шаг 3: Получение ключей

1. Перейди в "Настройки магазина"
2. Скопируй:
   - **ID магазина** - например: `12345`
   - **Секретное слово 1** - например: `abc123`
   - **Секретное слово 2** - например: `def456`

## Шаг 4: Настройка Result URL (webhook)

1. В настройках магазина найди "Result URL"
2. Вставь:
   ```
   https://auron-client-production.up.railway.app/api/v1/payment/freekassa-webhook
   ```
3. Выбери метод: POST
4. Сохрани

## Шаг 5: Добавление переменных в Railway

1. Зайди в Railway → auron-client → Variables
2. Добавь три новые переменные:
   - **FREEKASSA_MERCHANT_ID** = `твой ID магазина`
   - **FREEKASSA_SECRET_WORD_1** = `твое Секретное слово 1`
   - **FREEKASSA_SECRET_WORD_2** = `твое Секретное слово 2`
3. Нажми "Add" для каждой
4. Нажми "Restart" на последнем деплое

## Шаг 6: Включение методов оплаты

В личном кабинете Freekassa включи:
- ✅ Банковские карты
- ✅ QIWI Wallet
- ✅ ЮMoney
- ✅ СБП

## Шаг 7: Проверка

1. Открой сайт: `https://auron-client-production.up.railway.app/purchase.html`
2. Выбери тариф
3. Выбери "Банковская карта"
4. Нажми "Оплатить"
5. Должна открыться страница оплаты Freekassa
