# Настройка постоянного хранилища на Railway

Чтобы база данных не сбрасывалась при каждом деплое, нужно настроить Volume на Railway:

## Шаги:

1. Зайди в панель Railway: https://railway.app
2. Открой свой проект `auron-client-production`
3. Нажми на сервис (service)
4. Перейди в раздел "Variables"
5. Добавь переменную окружения:
   - Name: `DATA_DIR`
   - Value: `/app/data`

6. Перейди в раздел "Settings"
7. Найди секцию "Volumes"
8. Нажми "Add Volume"
9. Заполни:
   - Mount Path: `/app/data`
   - Size: 1GB (или больше если нужно)
10. Нажми "Add"

11. Railway автоматически пересоберет проект

После этого база данных будет сохраняться между деплоями!

## Альтернатива: PostgreSQL

Если хочешь более надежное решение, можешь использовать PostgreSQL:
1. В Railway нажми "New" → "Database" → "Add PostgreSQL"
2. Railway автоматически создаст переменные окружения
3. Нужно будет изменить код для работы с PostgreSQL вместо SQLite

Но для начала Volume с SQLite будет достаточно.
