# Auron Client - Minecraft 1.21.4

Официальный сайт для Auron Client - лучшего чита для Minecraft 1.21.4

## Особенности

- ✨ Современный дизайн с видео-фоном
- 🎮 Dynamic Island с событиями серверов
- 🎵 Музыкальный трекер
- 🔧 Поддержка модов
- 💎 Система подписок и тарифов
- 👤 Личный кабинет пользователя
- 🔐 Безопасная аутентификация

## Установка

1. Установите зависимости:
```bash
npm install
```

2. Создайте файл `.env` с настройками:
```
JWT_SECRET=your_secret_key
PORT=8081
```

3. Запустите сервер:
```bash
npm start
```

Сайт будет доступен по адресу: `http://localhost:8081`

## Структура проекта

- `/img/` - изображения и медиа
  - `/img/background-video/background.mp4` - видео-фон
  - `/img/logo.png` - логотип Auron Client
- `/css/` - стили
- `/js/` - JavaScript файлы
- `/api/` - API эндпоинты
- `server.js` - основной сервер

## Технологии

- Node.js + Express
- SQLite база данных
- JWT аутентификация
- Vanilla JavaScript
- CSS3 с backdrop-filter
