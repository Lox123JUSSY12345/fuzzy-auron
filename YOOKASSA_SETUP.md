# Настройка ЮKassa для оплаты картами и QIWI

## Шаг 1: Регистрация в ЮKassa

1. Зайди на https://yookassa.ru/
2. Нажми "Подключить ЮKassa"
3. Зарегистрируйся (нужны документы ИП или ООО)
4. После регистрации получишь доступ к личному кабинету

## Шаг 2: Получение API ключей

1. Зайди в личный кабинет ЮKassa
2. Перейди в раздел "Настройки" → "Настройки магазина"
3. Найди:
   - **shopId** (ID магазина) - например: `123456`
   - **Секретный ключ** - например: `live_abc123def456...`

## Шаг 3: Настройка webhook

1. В личном кабинете ЮKassa перейди в "Настройки" → "Уведомления"
2. Добавь URL для уведомлений:
   ```
   https://auron-client-production.up.railway.app/api/v1/payment/yookassa-webhook
   ```
3. Выбери события: "Успешный платеж" (payment.succeeded)
4. Сохрани настройки

## Шаг 4: Добавление переменных в Railway

1. Зайди в Railway → auron-client → Variables
2. Добавь две новые переменные:
   - **YOOKASSA_SHOP_ID** = `твой shopId`
   - **YOOKASSA_SECRET_KEY** = `твой секретный ключ`
3. Нажми "Restart" на последнем деплое

## Шаг 5: Включение методов оплаты

В личном кабинете ЮKassa включи:
- ✅ Банковские карты (Visa, Mastercard, МИР)
- ✅ QIWI Wallet
- ✅ Другие методы по желанию (СБП, ЮMoney и т.д.)

## Тестирование

Для тестирования используй тестовые карты:
- **Успешная оплата**: `5555 5555 5555 4477`
- **Отклоненная оплата**: `5555 5555 5555 5559`
- Срок: любая будущая дата
- CVC: любые 3 цифры

## Важно!

⚠️ ЮKassa требует:
- Юридическое лицо (ИП или ООО)
- Договор с ЮKassa
- Комиссия ~2.8% + 10₽ за транзакцию

## Альтернатива (если нет ИП/ООО)

Если у тебя нет ИП/ООО, можешь использовать:
1. **Robokassa** - принимает физлиц
2. **Freekassa** - без документов, но выше комиссия
3. **Cryptomus** - криптовалюта без документов
